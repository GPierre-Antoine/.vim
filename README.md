# vimrc
My own vimrc

```bash
git clone git@github.com/GPierre-Antoine/vimrc.git
ln -s vimrc/.vimrc ~/.vimrc
```


---

## quit
* quit with C-D
* force quit with C-S-D

## fold
* fold more with M-Down
* fold less with M-Up
* toggle fold with Space

## change
* revert with M-Left
* redo   with M-Right
